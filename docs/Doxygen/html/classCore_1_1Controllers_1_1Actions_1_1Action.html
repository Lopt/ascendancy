Detailed Description

Action class. Should be used as base for each other action. Actions are part of the game logic, which can be used from the server or the client to manipulate the world. Every action should do deterministic calculations, so the client can calculate the current game state by an old game state and the list of actions which were executed since the last loaded game state.
